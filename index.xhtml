com.io7m.ieee754b16

The ieee754b16 package implements a set of functions for converting to and from the IEEE 754 binary16 type.

Features

  • High coverage test suite (100%, minus an unreachable private constructor)
  • Fully documented
  • OSGi-ready
  • JPMS-ready
  • ISC license
